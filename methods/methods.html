Why do we add functions to the object and not jsut keep them separate
functions can be named the same but specifc to a object. Such as
var dog = {};
dog.speak{
return "woof!";
}
var cat = {};
cat.speak{
return "Meow!";
}
The function speak has a specific value to return depending on wich function it is in.
This also avoids namespace collisions wich would be having to versions of
the function speak outside of a object. the last value would be the only
one returned when the function is called.
function speak {
return "Meow";
}
Function speak{ return "Woof!";
}
In this case the only value for speak(); would be "Woof!"
